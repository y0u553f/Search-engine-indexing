this is Search engine indexing for text files .
formulas used :
tf=nombre d'occurence d'un mot dans un document
idf=log(|d|/Dj,tj app Dj)
poid mot dans un document= tf*idf;
poid mot dans un requette : si mot exist dans la liste des mot dist alors poid du mot = 0.5;
